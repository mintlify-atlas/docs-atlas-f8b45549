Core UI components used throughout the Plotlist app. All components are built with React Native and styled using NativeWind (Tailwind CSS).

## UI Components

### Avatar

Displays a user avatar image or initials fallback.

```tsx components/Avatar.tsx
export function Avatar({
  uri,
  label,
  size = 40,
}: {
  uri?: string | null;
  label?: string | null;
  size?: number;
})
```

**Props:**
- `uri` - Avatar image URL
- `label` - Name to generate initials from (used as fallback)
- `size` - Avatar diameter in pixels (default: 40)

**Usage:**
```tsx
<Avatar 
  uri={user.avatarUrl} 
  label={user.displayName} 
  size={44} 
/>
```

### Poster

Displays a show/movie poster with aspect ratio 2:3.

```tsx components/Poster.tsx
export function Poster({
  uri,
  size = "md",
  width,
  className,
}: {
  uri?: string | null;
  size?: "sm" | "md" | "lg";
  width?: number;
  className?: string;
})
```

**Props:**
- `uri` - Poster image URL
- `size` - Preset size (sm: 60px, md: 90px, lg: 140px)
- `width` - Custom width in pixels (overrides size)
- `className` - Additional Tailwind classes

**Usage:**
```tsx
<Poster uri={show.posterUrl} size="lg" />
```

### StarRating

Interactive 5-star rating component with half-star support.

```tsx components/StarRating.tsx
export function StarRating({
  value,
  onChange,
  size = 28,
  readonly = false,
}: {
  value: number;
  onChange?: (value: number) => void;

  readonly?: boolean;

```

**Props:**
- `value` - Current rating (0-5, supports 0.5 increments)
- `onChange` - Callback when rating changes
- `size` - Star icon size in pixels
- `readonly` - Disable interaction

**Usage:**
```tsx
<StarRating 
  value={rating} 
  onChange={setRating} 
  size={32} 
/>
```

## Design Patterns

### Haptic Feedback

All interactive components use haptic feedback:

```tsx
import * as Haptics from "expo-haptics";

const handlePress = () => {
  Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Light);
  // ... handle action
};
```

### Navigation

Components use `expo-router` for navigation:

```tsx
import { useRouter } from "expo-router";

const router = useRouter();
router.push(`/show/${showId}`);
```

### Styling

All components use NativeWind (Tailwind) classes:

```tsx
<View className="rounded-2xl border border-dark-border bg-dark-card p-4">
  <Text className="text-base font-semibold text-text-primary">
    {title}
  </Text>
</View>
```

### Color Tokens

Common color classes used throughout:
- `bg-dark-card` - Card background
- `bg-dark-elevated` - Elevated surface
- `bg-dark-hover` - Active/hover state
- `border-dark-border` - Border color
- `text-text-primary` - Primary text
- `text-text-secondary` - Secondary text
- `text-text-tertiary` - Tertiary/muted text
- `bg-brand-500` - Brand blue
- `bg-status-danger` - Error/danger state
