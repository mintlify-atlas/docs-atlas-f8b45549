## Overview

The Shows API manages the TV show catalog, integrates with TMDB for metadata, implements caching for performance, and provides search capabilities.

## Queries

### get

Get a show by ID.

```typescript
export const get = query({
  args: { showId: v.id("shows") },
  handler: async (ctx, args) => {
    return await ctx.db.get(args.showId);
  },
});
```

<ParamField path="showId" type="Id<'shows'>" required>
  Show document ID
</ParamField>

**Returns:** Show document or `null`.

### getByExternal

Lookup show by external source and ID.

```typescript
export const getByExternal = query({
  args: { 
    externalSource: v.string(), 
    externalId: v.string() 
  },
  handler: async (ctx, args) => { ... },
});
```

<ParamField path="externalSource" type="string" required>
  External data source (typically "tmdb")
</ParamField>

<ParamField path="externalId" type="string" required>
  TMDB show ID
</ParamField>

**Returns:** Show document or `null`.

### search

Search shows using full-text search.

```typescript
export const search = query({
  args: { 
    text: v.string(), 
    limit: v.optional(v.number()) 
  },
  handler: async (ctx, args) => { ... },
});
```

<ParamField path="text" type="string" required>
  Search query
</ParamField>

<ParamField path="limit" type="number">
  Maximum results (1-50, default 20)
</ParamField>

**Returns:** Array of show documents matching the query.

## Actions

### searchCatalog

Hybrid semantic + lexical search using embeddings.

```typescript
export const searchCatalog = action({
  args: { text: v.string() },
  handler: async (ctx, args) => {
    return await ctx.runAction(api.embeddings.searchShows, {
      text: args.text,
      limit: 12,
    });
  },
});
```

<ParamField path="text" type="string" required>
  Natural language search query
</ParamField>

**Returns:** Array of shows ranked by hybrid relevance score.

**Features:**
- Semantic understanding ("shows like Breaking Bad but funny")
- Exact and prefix title matching
- Genre intent detection
- Popularity weighting
- TMDB fallback for missing shows

**Episode fields:**
- `episodeNumber` - Episode number
- `name` - Episode title
- `overview` - Episode description
- `airDate` - Air date
- `runtime` - Duration in minutes
- `stillPath` - Episode still image URL
- `voteAverage` - TMDB rating
- `crew` - Directors and writers (up to 4)
- `guestStars` - Guest stars (up to 5)

**Caching:**
- Recent seasons (aired within 6 months): 7 days
- Archive seasons: 180 days

**Rate limiting:** 30 requests per minute per user.

## Caching Strategy

The Shows API uses multiple cache tables:

### tmdbDetailsCache

Caches extended show details and season information.

**TTL:**
- Show details: 7 days
- Recent seasons: 7 days
- Archive seasons: 180 days

### tmdbSearchCache

Caches TMDB search results for 24 hours.

### tmdbListCache

Caches curated list results for 24 hours.

## Example Usage

### Search Shows

```typescript
import { useQuery } from "convex/react";
import { api } from "../convex/_generated/api";

function ShowSearch({ query }: { query: string }) {
  const results = useQuery(
    api.shows.search,
    query ? { text: query, limit: 20 } : "skip"
  );
  
  return (
    <div>
      {results?.map(show => (
        <ShowCard key={show._id} show={show} />
      ))}
    </div>
  );
}
```

### Get Show Details

```typescript
import { useAction } from "convex/react";
import { api } from "../convex/_generated/api";

function ShowDetails({ tmdbId }: { tmdbId: string }) {
  const [details, setDetails] = useState(null);
  const getDetails = useAction(api.shows.getExtendedDetails);
  
  useEffect(() => {
    getDetails({ externalId: tmdbId }).then(setDetails);
  }, [tmdbId]);
  
  if (!details) return <Loading />;
  
  return (
    <div>
      <h1>{details.tagline}</h1>
      <p>Seasons: {details.numberOfSeasons}</p>
      <WatchProviders providers={details.watchProviders} />
      <Cast cast={details.cast} />
    </div>
  );
}
```

### Import Show

```typescript
import { useAction } from "convex/react";
import { api } from "../convex/_generated/api";

function AddToDatabase({ tmdbId }: { tmdbId: string }) {
  const ingest = useAction(api.shows.ingestFromCatalog);
  
  const handleAdd = async () => {
    const showId = await ingest({
      externalSource: "tmdb",
      externalId: tmdbId,
    });
    console.log("Show added:", showId);
  };
  
  return <button onClick={handleAdd}>Add to Database</button>;
}
```
