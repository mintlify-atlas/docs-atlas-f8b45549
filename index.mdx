## Key Features

- **Episode-level tracking**: Log individual episodes with optional notes and ratings
- **Reviews and ratings**: Rate and review shows or individual episodes with spoiler warnings
- **Lists**: Curate public or private lists of shows with drag-and-drop reordering
- **Contact sync**: Find friends already using Plotlist through privacy-preserving contact matching
- **Taste profiles**: AI-generated taste profiles and similar-user matching based on viewing patterns
- **TMDB integration**: Rich show metadata, cast information, trailers, and provider links

## Built With

Plotlist is built on a modern mobile-first stack:

- **Expo + React Native** for cross-platform mobile development
- **Convex** for real-time database, queries, mutations, and serverless functions
- **TMDB API** for comprehensive TV show metadata and catalog
- **Gemini embeddings** for AI-powered recommendations and semantic search
- **Twilio Verify** for secure phone-based authentication
